During the course of Pelipaja 23k I worked with the following 4 games:

Neon Pinball

https://sanna-juoperi.itch.io/neon-pinball

- With Neon Pinball I worked with game design, menus, UI and code. I also created the level editor, which used OpenAI API as a component.

- The level editor allowed the player to create their own graphics to the pinball table via text based commands, which the game sent through to OpenAI API.

- After processing the commands, the game received the created image from OpenAI and set it to the table elements the player had selected. The game also dealt with any errors when sending or receiving the API call.

Cat Brain Stimulator

https://anttikarjalacentriafi.itch.io/cat-brain-stimulator

- Cat Brain Stimulator was a mobile centric game, which had cats as target audience.

- I participated in the game design and created the music and sound effects for the game. I also designed and implemented the UI and code for all the menus and audio, including audio level controls.

Jinglebells Rock!

https://anttikarjalacentriafi.itch.io/folly-jolly-xmash

- Jinglebells Rock! was a real brainchild of mine, where I destined to create an isometric shooter similar to "The Ascent" but in a humorous Christmas setting.

- I designed the game mechanics and wrote the game code. I also built the game environment and created the UI with a scoreboard and timer.

- The original design included boss rounds, powerups, dynamically changing environment and multiple different projectiles, as well as an intro and an outro animation.

- All the design features, including detailed graphics and fluent controls, are still being worked upon and I hope to finish them eventually.

Military Animals

https://sanna-juoperi.itch.io/military-animals

- Military Animals was designed as an Age of War -clone with cute animal figures.

- My primary responsibility was to create the character and environment animations for the game.

- I also took part in creating the documentation and game design, and in the end in writing code too. I also worked on keeping the production running by attending meetings and communicating with the team.
